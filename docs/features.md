# Features

## Sprite Editor

- **Canvas Manipulation**: Draw on a variable-size canvas.
- **Storage**: Save sprites as PNGs (backed by matrix data).
- **Colors**: Primary and secondary color selection with palette support.
- **Animation**: create multi-frame animations.
- **Tools**: Hover effects, standard assets.

## Map Editor

- **Canvas**: Large area for level design.
- **Tools**: Fill, Rectangle, Circle, Brush size control.
- **Undo/Redo**: Support for `Ctrl+Z`.
- **Layers**: Multi-layer support for complex maps.
- **Preview**: Minimap/Preview for navigation.
- **Asset Toolbar**: Drag and drop assets onto the map.

## Navigation & Project Management

- **Project Structure**: Organize work into Projects, Folders, and Files.
- **File Management**: Create, Rename, Delete, and Move files/folders.
- **Context Awareness**: Double-click assets to open in the appropriate editor (Sprite vs Map).
